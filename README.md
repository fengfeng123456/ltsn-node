# Node LTS 查看工具

利用 node.js 实现的 Node LTS 查看工具。

## 安装

```sh
npm install ltsn-node -g
```

## 启动项目

```sh
ltsn version # version 版本号，可以是 10 
ltsn -v # 显示版本号
ltsn -h # 显示帮助信息
```

## 文件夹结构

```sh
.
|-- README.md
|-- bin
|   |-- ltsn # 脚本文件 
|-- index.js # 暴露 libs 下的模块
|-- libs # 功能代码目录 
|   |-- query.js # 对数据进行格式化
|   |-- update.js # 获取和更新数据源
|-- node_modules
|-- package-lock.json
|-- package.json
```
